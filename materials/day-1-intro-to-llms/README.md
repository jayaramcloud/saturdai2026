# Day 1 — Intro to LLMs (10 of 10 SVGs)

Ten 1600×900 slide-style SVGs covering the big-picture AI/LLM landscape for
beginners, from a user's point of view. Built to match the site's dark
purple theme (see `src/app/globals.css`) using a CVD-validated categorical
palette: blue `#5b7cfa`, teal `#0d9488`, amber `#d97706`, magenta `#ec4899`,
green `#16a34a`, violet `#8b5cf6`.

Rendered on the site at `/week-1` via `Slideshow` (`src/components/Slideshow.tsx`),
served from `public/slides/week-1/`.

1. **01-ai-landscape.svg** — nested zones: AI → ML → Deep Learning →
   Generative AI → LLMs, with a "you are here" pin and familiar examples.
2. **02-llm-assistants-map.svg** — quadrant map of ChatGPT, Claude, Gemini,
   Copilot, Perplexity, Meta AI, Grok, positioned by general strength.
3. **03-day-in-the-life.svg** — six everyday use-case regions (writing,
   learning, coding, planning, creativity, advice) with example prompts.
4. **04-conversation-loop.svg** — the prompt → read → respond → refine loop,
   plus plain-language callouts for system instructions and context window.
5. **05-learning-roadmap.svg** — the bootcamp path: Day 1 (LLMs) → Week 2
   (RAG) → Week 3 (MCPs) → Week 4 (Tool Calling).
6. **06-prompting-basics.svg** — vague vs. specific prompt, side by side,
   with the resulting difference in output quality.
7. **07-failure-modes.svg** — three failure modes: hallucination, stale
   knowledge, and confidently wrong answers, each with a guard.
8. **08-privacy-and-data.svg** — where a typed message goes, and what
   categories of data are safer left out of a chat.
9. **09-free-vs-paid.svg** — comparison table of free vs. paid tiers across
   limits, model access, memory, uploads, and features.
10. **10-recap-and-exercise.svg** — five-point recap plus a three-step
    live exercise (vague prompt → rewrite → compare).

Open any file directly in a browser to preview, or view in an SVG/vector
editor (Figma, Illustrator, Inkscape) to fine-tune.
